# To Run Locally

```
git clone https://github.com/434huzaifa/FilmFolio
```

```
cd FilmFolio
```

add .env.local

```
# Either live link or local link without last /
VITE_BACK_END_URL=
```

```
npm i
```

```
npm run dev
```

enjoy
